Tentang CloudCast

CloudCast adalah aplikasi web untuk menampilkan informasi cuaca terkini dan prakiraan cuaca harian di berbagai kota. Dibangun untuk memberikan informasi cuaca yang mudah diakses, cepat, dan akurat.

Teknologi yang Digunakan

  • 🚀 Backend: Go (Golang)
  • 🎨 Frontend: HTML, Tailwind CSS
  • 🔍 Data Cuaca: Database Internal + API OpenWeather

Tujuan

Memberikan pengalaman terbaik dalam memantau cuaca harian dan mingguan, baik untuk perencanaan perjalanan, aktivitas luar ruangan, maupun keperluan lainnya.
